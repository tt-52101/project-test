Feather Icons

Huro is bundled with the Feather Icons javascript svg library. Everyone likes Font Awesome icons and they are widely used accross a large variety of websites. Please refer to the code example for more details about usage.

<i data-feather="activity"></i>
<!--Feather icon usage (js) -->
<script> feather.replace(); </script>
