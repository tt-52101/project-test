Basic Cards

Huro cards are very versatile and can be used in any type of layout. Huro provides 3 main basic cards with the following classes: .s-card, .r-card and .l-card. The main difference each one of those is the border radius, giving a unique look and feel to each one of them. If you explore Huro SCSS code, you will notice that they serve as a base for numerous components, using the SCSS @extend directive.

Iam an S-Card

I can be used as is in any layout. S Cards are simple containers that can hold any type of content, from simple plain text to more structured markup.

Elevated Cards

Huro cards can be elevated and show a slight box shadow. To display elevated cards, simply add the is-raised class to the target card element.
