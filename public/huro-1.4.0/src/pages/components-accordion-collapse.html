Collapse

Huro provides a simple to use collapse component. There are 2 available variations that you can use to change the collapse header icon. Add the has-plus class to the collapse-icon element to use a plus icon. Add the has-chevron class to the collapse-icon element to use a chevron icon. Check markup for more details. 


<div class="collapse has-plus">
    <div class="collapse-header">
        <h3>Plus Item 1</h3>
        <div class="collapse-icon">
            <i data-feather="plus"></i>
        </div>
    </div>
    <div class="collapse-content">
